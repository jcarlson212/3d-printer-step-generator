# Authoritative sources

The `.md` files in this folder are **distilled** reference notes authored for this
project (concise + token-efficient for prompt injection, no licensing baggage).
For the full upstream material, see below. `scripts/fetch_resources.py` can pull
the openly-licensed ones into `resources/downloaded/` (git-ignored) if you want
local copies; we deliberately do not vendor large or copyrighted docs.

## CAD / build123d (open, safe to fetch)
- build123d docs: https://build123d.readthedocs.io/en/latest/
- build123d cheat sheet: https://build123d.readthedocs.io/en/latest/cheat_sheet.html
- build123d API: https://build123d.readthedocs.io/en/latest/direct_api_reference.html
- OpenCascade (kernel behind it): https://dev.opencascade.org/doc/overview/html/

## STEP / STL formats
- STEP / ISO 10303 overview: https://en.wikipedia.org/wiki/ISO_10303
- STEP AP242: https://www.ap242.org/
- STL format: https://en.wikipedia.org/wiki/STL_(file_format)

## FDM / PLA printing
- Bambu Lab A1 mini specs & wiki: https://wiki.bambulab.com/en/a1-mini
- Bambu PLA Basic settings: https://wiki.bambulab.com/en/filament-acc/filament/bambu-pla-basic
- Design rules for FDM (overhangs/bridges/walls): vendor design guides (e.g. Prusa,
  Bambu, Hubs "knowledge base / design rules for 3D printing").

## Staunton knight / Parthenon heritage
- Staunton chess set: https://en.wikipedia.org/wiki/Staunton_chess_set
- Elgin Marbles / Parthenon frieze (Selene's horse): https://en.wikipedia.org/wiki/Elgin_Marbles
- British Museum Parthenon collection: https://www.britishmuseum.org/collection
- Thingiverse / Printables: search "Staunton knight" for printable reference models.
